Model overview

This is the interactive companion to the NP-MT-RNM regulatory-network model described in Workineh & Noailly (2026). The simulation engine is tellurium, driven by the SBML export of the network used in the paper. Use the Network tab to explore the topology, and the Simulation tab to run the model live with custom boundary regimes and node clamps.
